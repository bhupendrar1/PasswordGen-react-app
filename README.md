# PasswordGen-react-app

A simple, accessible, and customizable **Password Generator** built with React. Generate strong, secure passwords with options for length and character types (uppercase, lowercase, numbers, symbols). Perfect for learning React project structure and building a small utility app.

---

## Features

* Generate secure passwords with configurable length.
* Toggle inclusion of uppercase letters, lowercase letters, numbers, and symbols.
* Copy generated password to clipboard with one click.
* Password strength indicator.
* Responsive UI built with modern CSS / Tailwind (optional).
* Small, well-documented codebase — great for beginners.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/bhupendrar1/PasswordGen-react-app.git
cd PasswordGen-react-app
```

2. Install dependencies:

```bash
# with npm
npm install

# or with yarn
yarn install
```

3. Start the development server:

```bash
npm start
# or
yarn start
```

The app will open at `http://localhost:3000` by default.

## Usage

1. Use the slider or number input to select the desired password length.
2. Toggle the checkboxes for including uppercase, lowercase, numbers, and symbols.
3. Click `Generate` to create a password.
4. Click the copy icon/button to copy the password to your clipboard.
5. The password strength indicator updates based on selected options and length.
